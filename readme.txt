RentalForge is a WooCommerce WordPress rental plugin for property & equipment booking. Manage bookings, pricing, availability & orders.

== Description ==

RentalForge is a powerful WooCommerce-based WordPress rental plugin for managing property rentals and equipment rentals from a single dashboard. It helps you handle bookings, availability, pricing, and rental orders with a streamlined and efficient workflow.

This WordPress rental plugin supports flexible pricing options including hourly, daily, and monthly rentals.

You can easily configure booking rules, set minimum and maximum rental durations, manage deposits, and control availability using schedules or custom date settings.

 RentalForge is built to be lightweight yet feature-rich, making it an ideal rental booking plugin for both developers and business owners. It eliminates the complexity of traditional eCommerce setups while still providing advanced rental management features.

 Whether you are running a property rental business, equipment rental service, or both, RentalForge gives you full control over your inventory, bookings, and pricing.

  It is perfect for managing vacation rentals, construction tools, event equipment, vehicles, electronics, and more. With its user-friendly interface and powerful backend, this WordPress booking and rental plugin allows you to manage everything directly from your dashboard.

== Installation ==

= Automatic Installation (Recommended) =

1. Go to **Plugins → Add New** in your WordPress admin panel.
2. Search for **RentalForge**.
3. Click **Install Now** and then **Activate**.

= Manual Installation via Upload =

1. Upload the abprf-rental-forge.zip folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Look for the "RentalForge" menu in your admin sidebar to start configuring your equipment/Property.

= Using FTP =

1. Download abprf-rental-forge.zip
2. Extract the abprf-rental-forge directory to your computer
3. Upload the abprf-rental-forge directory to the /wp-content/plugins/ directory
4. Activate the plugin in the Plugin dashboard

== Usage ==

1. Go to **RentalForge → Add Property**
2. Add title, description, and images
3. Configure pricing (at least one required)
4. Set rental rules (min/max duration)
5. Configure deposit (optional)
6. Set availability (schedule or special dates)
7. Use shortcode below to display properties
1. Go to **RentalForge → Settings** and complete the initial setup.
2. Add your first property under **RentalForge → Properties → Add New**.
3. Place `[abprf-post]` on any page to display your listings.

== Frequently Asked Questions ==

= Can I use multiple pricing types? =
Yes, you can enable one or multiple pricing options.

= Is deposit required? =
No, deposit is optional.

= What is the default minimum quantity? =
If not set, the minimum quantity will be 1.

= Can I override off days or weekends? =
Yes, using special on dates, you can force availability.

= How is monthly duration calculated? =
By default, 1 month = 30 days .

= Does the plugin work with any WordPress theme? =

Yes! RentalForge is designed to work with any well-coded WordPress theme. It comes with its own responsive frontend templates that can change style from configuration .

= Which payment gateways are supported? =

RentalForge uses the WooCommerce payment system, so all transactions are processed through your configured WooCommerce payment gateways.

= Can I manage multiple properties? =

Absolutely. You can add unlimited properties, each with its own gallery, pricing, availability calendar and booking settings.

= Does it support automatic and manual booking approval? =

Yes. You can choose **Instant Booking** (auto-confirmed) or **Manual Approval** per property, or set a global default in Settings.

= Is it translation ready? =

Yes. The plugin is fully translation-ready.

= Where can I get help? =

Post your question in the [support forum](https://wordpress.org/support/plugin/rentalforge/) and our team will respond within 24–48 hours.

= Does it work with XAMPP? =
Yes, RentalForge is fully compatible with local environments like XAMPP for development and testing.
