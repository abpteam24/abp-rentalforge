RentalForge is a WooCommerce WordPress rental plugin for property & equipment booking. Manage bookings, pricing, availability & orders.

== Description ==

RentalForge is a powerful WooCommerce-based WordPress rental plugin for managing property rentals and equipment rentals from a single dashboard. It helps you handle bookings, availability, pricing, and rental orders with a streamlined and efficient workflow.

This WordPress rental plugin supports flexible pricing options including hourly, daily, and monthly rentals.

You can easily configure booking rules, set minimum and maximum rental durations, manage deposits, and control availability using schedules or custom date settings.

 RentalForge is built to be lightweight yet feature-rich, making it an ideal rental booking plugin for both developers and business owners. It eliminates the complexity of traditional eCommerce setups while still providing advanced rental management features.

 Whether you are running a property rental business, equipment rental service, or both, RentalForge gives you full control over your inventory, bookings, and pricing.

  It is perfect for managing vacation rentals, construction tools, event equipment, vehicles, electronics, and more. With its user-friendly interface and powerful backend, this WordPress booking and rental plugin allows you to manage everything directly from your dashboard.

== Shortcodes ==

Use these shortcodes anywhere on your site:

[abprf-booking]      — Display all post listings and booking
[abprf-post]      — Display all post listings
[abprf-property]  — Display all property listings
[abprf-gallery]  — Display all property/post image

== Payment Gateways ==

RentalForge does not include a separate payment system. All payments are fully managed by WooCommerce and its supported payment gateways.

== Theme Compatibility ==

RentalForge is compatible with any well-coded and standards-compliant WordPress theme. It includes responsive frontend templates that adapt to different layouts.

== Installation ==

= Automatic Installation (Recommended) =

1. Go to Plugins → Add New in your WordPress dashboard.
2. Search for "RentalForge".
3. Click Install Now and then Activate.

= Manual Installation via Upload =

1. Upload the abp-rentalforge.zip file to the /wp-content/plugins/ directory.
2. Activate the plugin from the Plugins menu in WordPress.
3. Access the "RentalForge" menu from the admin sidebar to start configuration.

= Installation via FTP =

1. Download the abp-rentalforge.zip file.
2. Extract the abp-rentalforge folder on your computer.
3. Upload the folder to /wp-content/plugins/ via FTP.
4. Activate the plugin from the WordPress Plugins dashboard.

== Usage ==

1. Go to RentalForge → Settings and complete the initial setup.
2. Add your first property under RentalForge → Properties → Add New.
3. Configure title, description, and images.
4. Set pricing (at least one pricing type is required).
5. Configure rental rules (minimum/maximum duration).
6. Set deposit and availability options if needed.
7. Use the shortcode below to display listings on any page:

   [abprf-post]

== Frequently Asked Questions ==

= Can I use multiple pricing types? =
Yes. You can enable multiple pricing options, but only one pricing rule can be active per post.

= Is a deposit required? =
No. Deposit is completely optional.

= What is the default minimum quantity? =
If not set, the default minimum quantity is 1.

= Can I override unavailable days or weekends? =
Yes. Special date settings allow you to override default availability rules.

= How is monthly duration calculated? =
By default, 1 month is calculated as 30 days.

= Does the plugin work with any WordPress theme? =
Yes. RentalForge is designed to work with any well-coded WordPress theme and includes responsive frontend templates.

= Which payment gateways are supported? =
RentalForge uses WooCommerce for payments, so all WooCommerce-supported gateways are available.

= Can I manage multiple properties? =
Yes. You can create unlimited properties, each with individual pricing, availability, and settings.

= Does it support automatic and manual booking approval? =
Yes. You can choose between Instant Booking (auto-approved) or Manual Approval mode.

= Is it translation ready? =
Yes. The plugin is fully translation-ready and compatible with standard WordPress localization tools.

= Does it work in local environments like XAMPP? =
Yes. RentalForge works perfectly in local development environments such as XAMPP or LocalWP.
